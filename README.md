# rl-highway-lab
高速道路を走る車両の挙動を強化学習で学習させます  
AIで全部コード書かせようとしたやつがあるが  
何も理解してなくてわかんなくなったので作り直すことにした   

# 強化学習の流れ 
**1.初期化**  

**2.学習ループ**  
&emsp;2-1. シミュレーション  
&emsp;2-2. 報酬計算  
&emsp;2-3. 観測  
&emsp;2-4. 行動を計算する  
   
 行動を計算する部分はニューラルネットワークとやらゆうところになるが  
 その辺はパッケージ使えばいい感じにやってくれるらしい  
 実装すべき部分はシミュレーションと報酬設定
